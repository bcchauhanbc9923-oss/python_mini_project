
Bank Management System
A Graphical User Interface (GUI) based application built using Python and Tkinter that simulates a basic bank management system. It enables users to manage bank accounts efficiently, including adding, updating, deleting, depositing, withdrawing, and checking balances through an intuitive interface.

Features
•	Add New Accounts: Create and store customer account details such as Account Number, Name, Account Type, and Balance.
•	Update Records: Modify existing account information easily.
•	Delete Accounts: Remove customer records instantly.
•	Deposit and Withdraw: Perform secure deposit and withdrawal operations.
•	Check Balance: Quickly view the current balance of any selected account.
•	Data Validation: Ensures correct input and prevents invalid entries.
•	Interactive TreeView: Displays all stored account records in a tabular format.
•	Modern Interface: Designed using Tkinter’s widgets for a clean, professional look.
	
 How It Works
The application provides a virtual banking record manager using Python’s GUI framework.
Technologies Used
•	Tkinter: For building the graphical user interface.
•	ttk (Themed Widgets): For displaying customer records in a data table.
•	Messagebox: For showing alerts and success messages.
•	Python Core Logic: For performing all operations internally without a database.

Functional Overview
•	Add Account
•	Enter all required details and click Add to store a new record.
•	Update Account
•	Select a record, modify details, and click Update to apply changes.
•	Delete Account
•	Select a record and click Delete to remove it permanently.
•	Deposit Money
•	Select an account, enter an amount, and click Deposit to increase balance.
•	Withdraw Money
•	Select an account, enter an amount, and click Withdraw to decrease balance.
•	Check Balance
•	Select an account and click Check Balance to view account details.
•	Clear Fields
Click Clear to reset all entry fields.

Interface Layout
•	Left Panel – Account Details
Contains input fields and buttons for account operations.
•	Right Panel – Customer Records
Displays stored account data in a Treeview table.
